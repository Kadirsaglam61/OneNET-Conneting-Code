根据各位同志们的需求，将平时自己弄的一部分接入设备的源代码放到云端，如有需要请自行下载，希望对大家学习OneNET有所帮助，同时也欢迎大家多提意见。

源代码用到的硬件平台主要有两类：STM32和Arduino开发板，基本上在网上可以轻易购得，每个源码实例在论坛里面也可以找到相对应案例发帖，有兴趣的可以参考。

STM32源码开发环境：Keil uVision 4.54.0.0
Arduino源码开发环境：Arduino 1.6.1

PS：OneNET一直都在发展变化中，源代码中用到的一些OneNET接入SDK（例如EDP SDK）版本可能较旧（很
久以前写的），坛友们可以根据需求略作修改。
